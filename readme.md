# wiimote-rs

Rust library/driver for Nintendo Wii Remotes (aka. Wiimote).

## Supported Features

- Buttons
- Speaker
- Rumble
- Player LEDs
- IR Camera
- Accelerometer

## Not yet implemented features

- Extensions
  - Nunchuk
- Motionplus

See [`examples`](./examples).
