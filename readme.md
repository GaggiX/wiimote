# wiimote-rs

Rust library/driver for Nintendo Wii Remotes (aka. Wiimote).

## Supported Features

- Buttons
- Speaker
- Rumble
- Player LEDs
- IR Camera
- Accelerometer

## Not yet implemented features

- Status reports
- Extensions
  - Nunchuk
  - Motionplus
- Check compatibility of reporting mode and enabled peripherals or abstract that
  away completely
